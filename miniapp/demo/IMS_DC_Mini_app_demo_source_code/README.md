## Description
This project is a mini-program development example. Developers can refer to it to call the DC and other capability APIs provided by the SDK.

## Compilation
npm run build

## Packaging
Package the compiled files into a ZIP archive to create the IMS DC Mini app. The index.html and properties.json files must be placed in the root directory of the ZIP package.
For reference, see:[miniapp/demo/IMS_DC_Mini_app_demo.zip](../IMS_DC_Mini_app_demo.zip)
